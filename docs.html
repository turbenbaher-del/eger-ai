🎣 Егерь ИИ

Первый российский ИИ-помощник рыбака — прогноз клёва, дневник уловов, сообщество рыбаков и маркетплейс снастей в одном PWA-приложении. Работает на любом смартфоне без установки из App Store.

PWA · без App Store Firebase · Realtime React 18 · Babel Claude AI API Яндекс.Карты Open-Meteo Weather

Полный список функций

Все возможности приложения в текущей версии v2.0.0.
🌤️

Прогноз клёва

Алгоритм по давлению, ветру, температуре воды, сезону и фазе луны. Оценка 1–10 с дуговым индикатором BiteArc.

🤖

ИИ-советник (Claude API)

Отвечает на любой вопрос о рыбалке. Fallback на локальный бот с 25+ интентами при отсутствии интернета.

💬

Telegram-like чат

Reply, реакции, свайп для ответа, лонг-пресс меню, группировка сообщений, разделители дат, тики прочтения.

🔍

Поиск по чату

Живой поиск с подсветкой совпадений в тексте, именах, никнеймах. Счётчик результатов.

💌

Личные сообщения (DM)

Нажми на имя пользователя → открывается приватный чат в реальном времени через Firestore.

📸

Медиа в чатах

Фото из галереи, видео-кружки (60 сек), голосовые сообщения, загрузка в Firebase Storage.

📍

Точки рыбалки

60+ GPS-координат для 18 видов рыб. Сортировка по расстоянию от пользователя. Яндекс.Карты.

🗺️

Интерактивная карта

Яндекс.Карты с маркерами рыбных мест. Кластеризация, уровни воды, слои.

📰

Новости в реальном времени

Firestore onSnapshot + боты публикуют из Telegram-каналов. Анимация новых статей.

📝

Отчёты рыбаков

Публикация отчётов с фото, местом, видом рыбы. Лайки, редактирование, удаление.

🔔

Push-уведомления

FCM через ServiceWorker. Уведомления о новостях, хорошем клёве. Работает в фоне.

🏆

Турниры

Инструкция пользователя

Для всех рыбаков, использующих Егерь ИИ

Первые шаги

1

Открой приложение

Перейди на https://turbenbaher-del.github.io/eger-ai/ в браузере на телефоне или компьютере. Приложение загрузится автоматически — установка не нужна.

2

Войди через Google

Нажми на иконку профиля (👤) внизу → кнопка «Войти через Google». Используй любой Google-аккаунт. Авторизация занимает 5 секунд.

3

Разреши геолокацию

Браузер запросит доступ к местоположению — нажми «Разрешить». Это нужно чтобы ИИ находил ближайшие точки рыбалки к тебе и сортировал их по расстоянию.

4

Установи приложение (необязательно)

Chrome/Edge покажет кнопку «Установить» или «Добавить на главный экран». После этого приложение работает как нативное — без браузерной строки, в офлайн-режиме.

Главный экран — Прогноз

Главный экран — твоя сводка на сегодня. Здесь собраны все данные которые нужны перед выездом.

🎯

Прогноз клёва (1–10)

Оценка рассчитывается автоматически по давлению, ветру, температуре воды, сезону и фазе луны. 8–10 = отличный клёв, 5–7 = хороший, ниже 5 — лучше перенести.

🌡️

Текущая погода

Температура воздуха и воды, ветер, давление, влажность. Обновляется каждые 15 минут с Open-Meteo API. Давление 758–765 мм рт.ст. — лучший диапазон для клёва.

🌙

Фаза луны

Текущая фаза с советом: полнолуние = мирная рыба активна весь день, новолуние = хищник ночью. Обновляется в реальном времени.

📊

График давления

Sparkline-график давления по часам. Синяя точка — текущий момент. Резкие перепады = плохой клёв, стабильность = хорошо.

Совет: Листай вниз на главном экране — там 7-дневный прогноз с иконками погоды и вероятностью осадков. Идеально для планирования выезда на выходные.

ИИ-чат — советник Егерь

Нажми вкладку 🤖 ИИ внизу. Егерь отвечает на любой вопрос о рыбалке.

Что умеет ИИ-советник:

СпросиПолучишь
«Где ловить леща?»GPS-точки с ссылкой на Яндекс.Карты, расстояние от тебя
«Клюёт сегодня?»Прогноз клёва 1–10 с анализом погоды, луны и сезона
«Как ловить щуку осенью?»Снасти, насадки, техника, лучшие часы, точки
«Какое удилище купить?»Конкретный совет под вид рыбы и бюджет + ссылки на магазины
«Почему не клюёт?»Анализ давления, луны, температуры + советы что изменить
«Нахлыст для начинающих»Полное руководство: снасти, техника заброса, мушки

Функции чата:

↩ Ответить на сообщение

Свайп вправо на сообщении — появится иконка ↩, затем цитата в поле ввода. Или нажми и удерживай → меню «Ответить».

😊 Реакции

Нажми и удерживай → меню с 6 эмодзи (👍❤️😂😮🎣🔥). Выбери реакцию — она появится под сообщением.

🎤 Голосовое сообщение

Удерживай кнопку микрофона → говори → отпусти. Запись до 2 минут. Отмена — свайп влево при удержании.

📸 Отправить фото

Нажми кнопку прикрепления (+) → «Фото из галереи» или «Камера». ИИ может прокомментировать снимок.

Быстрые кнопки: Под полем ввода есть 8 быстрых вопросов — «Прогноз клёва», «Точки рыбалки», «Что ловить», «Луна», «Магазины» и др. Нажми чтобы сразу получить ответ без набора текста.

Чат сообщества

Вкладка 💬 Сообщество — общий чат всех рыбаков приложения в реальном времени.

Что можно делать:

💬

Написать сообщение

Текст, фото, видео-кружок (до 60 сек), голосовое. Используй кнопку (+) для медиа.

↩

Ответить на сообщение

Нажми на сообщение → появится меню. Выбери «↩ Ответить» — цитата подставится в поле ввода.

🔍

Поиск по чату

Нажми 🔍 в заголовке → вводи текст. Совпадения подсвечиваются зелёным. Показывает количество найденных сообщений.

💌

Написать личное сообщение

Нажми на имя/аватар пользователя → открывается личный чат (DM). История сохраняется. Доступна только вам двоим.

Правила чата: запрещена реклама, спам, оскорбления, нецензурная лексика. Нарушителей блокируют модераторы. Сообщай о нарушениях через функцию «Пожаловаться» (в меню сообщения).

Новости и отчёты

Вкладка 📰 Новости содержит два раздела: «Новости» и «Отчёты рыбаков».

Раздел «Новости»:

Официальные новости Росрыболовства, Рыбнадзора, данные об уровне воды. Новые материалы подсвечиваются красной полосой и анимацией. Включи push-уведомления (кнопка 🔔 в заголовке) — получай важные новости на телефон.

Раздел «Отчёты рыбаков»:

Отчёты пишут сами пользователи. Нажми «+ Новый отчёт» чтобы поделиться уловом с сообществом. Можно добавить фото, указать место и вид рыбы. Другие пользователи могут ставить лайки.

Проверка логов функций:

1

Firebase Console → Functions → Logs

Реальное время выполнения, ошибки, duration каждого вызова.

2

Google Cloud Console → Cloud Logging

Расширенный поиск: resource.type="cloud_function" AND resource.labels.function_name="runBots"

Правила Firestore

Файл: firestore.rules. Деплой: firebase deploy --only firestore:rules --project eger-ai

Важно: После любого изменения rules необходимо деплоить! Тестирование через Firebase Console → Firestore → Rules → «Тестирование правил».

Ключевые принципы применённых правил:

  • Пользователь читает/пишет только свои данные (isOwner(userId))
  • Публичные коллекции (сообщения, новости, отчёты) — только для авторизованных
  • Рейтинг, значки, AI-счётчики — только Cloud Functions (allow write: if false)
  • DM: участники /dms/{dmId}.participants[] имеют доступ к переписке
  • FCM-токены: только запись, никто не может читать чужие токены

Деплой приложения

1

Внести изменения в index.html

Все изменения UI — в файле C:\Users\user\Downloads\eger\index.html.

2

Закоммитить

git add index.html && git commit -m "описание изменений"

3

Запушить на GitHub Pages

git push origin master — через 1–3 минуты изменения появятся на сайте.

4

Деплой Firestore rules (если изменились)

$env:PATH = "C:\Program Files\nodejs;" + $env:PATH
firebase deploy --only firestore:rules --project eger-ai

5

Деплой Cloud Functions (если изменились)

firebase deploy --only functions --project eger-ai
Занимает 3–5 минут.

ОСТОРОЖНО: никогда не пушь файл .env или файлы с секретными ключами (API key Claude, сервисный аккаунт Firebase). Firebase конфиг в коде — допустимо, он публичный и защищён правилами Firestore. Секреты Claude API хранятся только в переменных окружения Cloud Functions.

Как удалить сообщение из Firestore:

1

Открой Firebase Console

console.firebase.google.com → проект eger-ai → Firestore Database

2

Перейди в коллекцию /messages

Найди нужное сообщение по полю text или uid автора. Можно использовать фильтр.

3

Удали документ

Нажми на ID документа → кнопка 🗑️ «Удалить документ» → подтверди. Сообщение исчезнет у всех в реальном времени.

В будущем: планируется добавить функцию «Пожаловаться» с автоматическим уведомлением модератора и инструмент модерации прямо в интерфейсе приложения без Firebase Console.

Модерация отчётов рыбаков

Отчёты публикуются в коллекции /reports. Правила для проверки:

КритерийДопустимоНедопустимо
ФотоУлов, рыбалка, снасти, местаНеприемлемый контент, личные данные людей
ТекстОписание рыбалки, советы, вопросыРеклама, оскорбления, спам
Место ловлиЛюбой водоём Ростовской области и РФКоординаты частной территории
Способ ловлиЛегальные методыЭлектроудочка, сети в запрещённых местах, взрывчатка

Как удалить отчёт:

Коллекция /reports в Firestore → найти по userId или title → удалить документ. Автор получает возможность опубликовать новый отчёт.

Работа с пользователями

Блокировка пользователя:

1

Firebase Console → Authentication → Users

Найди пользователя по email или UID.

2

Отключи аккаунт

Нажми на пользователя → «Отключить аккаунт» (Disable account). Пользователь не сможет войти и потеряет сессию.

3

Опционально: удали данные

Если необходимо — удали документ /users/{uid} в Firestore. Сообщения в чате останутся с именем, но без активного аккаунта.

Внимание: блокировка необратима в простом смысле — пользователь не сможет войти. Включить обратно: Firebase Console → Authentication → Users → тот же пользователь → «Включить аккаунт».
